### System Utility Commands

#### Commands covered

- `date`
- `uptime`
- `hostname`
- `uname`
- `which`
- `cal`
- `bc`

#### *date*

The `date` command returns information about the current date of the system. 
It returns the following information:

| `date`           |
|------------------|
| Day of the week  |
| Month            |
| Day of the month |
| Time             |
| Time zone        |
 | Year				|

```bash
~ $ date
Tue Dec 28 11:38:47 CET 2021
~ $ 
```
